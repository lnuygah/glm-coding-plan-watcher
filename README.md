# glm-coding-plan-watcher

> 浏览器自动化工具，监控 [GLM Coding Plan](https://www.bigmodel.cn/glm-coding) 指定套餐
> （计费周期 × Lite/Pro/Max）是否可购买；命中时记录日志、截图、保存 HTML 快照、发通知，
> 并默认点击「购买/订阅入口」后**暂停等待人工**完成支付。

⚠️ **本项目仅用于个人可用性监控与提醒**。使用者需自行遵守目标网站的服务条款。详见下方
[安全与合规边界](#安全与合规边界)。

> 文档将在实现完成后补全（安装、登录、配置、命令、抓取真实 DOM 校准等）。
